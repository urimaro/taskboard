## 使い方マニュアル

* かんたんな手順
    1. Todoの一番下のAdd欄に入力EnterするとTodoにタスクが追加される
    
    2. TodoのタスクをDoingにドラッグ&ドロップ、同様にDoingからDoneに移動可能
    
    3. タスク右の'×'を押すと削除される
